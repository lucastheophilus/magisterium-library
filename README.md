# Biblioteca del Magisterio de la Iglesia Católica

> Una biblioteca organizada del Magisterio de la Iglesia Católica en formato Markdown.

## Propósito

Este proyecto tiene como objetivo preservar, organizar y facilitar el acceso a los documentos del Magisterio de la Iglesia Católica mediante una colección estructurada en formato Markdown.

Su finalidad es ofrecer una herramienta de consulta y estudio que permita acceder a los documentos de forma sencilla, legible y fácilmente navegable.

## Alcance

La colección comprende, de manera progresiva, documentos promulgados por la Iglesia Católica, entre ellos:

- Encíclicas
- Constituciones apostólicas
- Exhortaciones apostólicas
- Cartas apostólicas
- Motu proprio
- Declaraciones
- Instrucciones
- Directorios
- Otros documentos del Magisterio

El proyecto abarca el Magisterio desde el pontificado de **León XIII** hasta la actualidad.

## Organización

Los documentos se organizan siguiendo criterios históricos y documentales.

La estructura principal del repositorio se divide por:

- Pontificado
- Organismos de la Santa Sede
- Concilios
- Documentos generales de consulta

Cada documento se presenta en formato Markdown para facilitar:

- La lectura.
- La búsqueda de texto.
- El control de versiones.
- La preservación digital.
- La reutilización con fines académicos y de estudio.

## Criterios editoriales

Se procura mantener la máxima fidelidad posible a las ediciones oficiales, limitando las intervenciones editoriales a aspectos relacionados con el formato y la estructura del texto.

Las modificaciones realizadas no alteran el contenido doctrinal de los documentos.

## Fuentes

Siempre que sea posible, los documentos se transcriben a partir de las publicaciones oficiales de la Santa Sede.

- https://www.vatican.va/

En aquellos casos en que un documento no se encuentre disponible en el sitio oficial, podrán utilizarse otras fuentes de reconocido valor documental, procurando siempre identificar su procedencia.
